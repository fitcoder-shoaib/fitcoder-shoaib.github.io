CHAPTER 01 // 2020

The Lockdown Catalyst & Scripting Origins

Behind closed doors during the global COVID-19 lockdown, a spark of curiosity was lit. With nothing but Notepad and default Windows command prompts, I began experimenting with basic `.vbs` and CMD scripts. What started as simple execution errors soon turned into a deep fascination with how operating systems, memory execution, and networks behave behind the user interface.

CHAPTER 02 // 2021 - 2022

The Cyber Wilderness & Linux Labs

Basic scripts soon felt limiting. I transitioned into Linux systems administration and security mechanisms. Setting up virtualized sandboxes, troubleshooting routing setups, and exploring penetration testing frameworks became a daily self-guided ritual. This period solidified my understanding of network automation, secure system patterns, and bash tooling.

CHAPTER 03 // 2023 - PRESENT

Engineering The Future @ MBCET

To combine my practical self-study with standard academic foundations, I enrolled in Computer Science Engineering at Mar Baselios College of Engineering and Technology (MBCET). Immersing myself in academic algorithms, I began collaborating on systems development, attending technical hackathons, and researching automated compliance designs.

CHAPTER 04 // CURRENT HORIZONS

Intelligence & Secure Infrastructures

Currently focusing on the intersection of cybersecurity and artificial intelligence. I build and test systems using Transformers, LSTMs, Vector Databases, and Retrieval-Augmented Generation (RAG) agent flows. At the same time, I study large-scale network structures including MPLS and autonomous network threat prevention systems.
